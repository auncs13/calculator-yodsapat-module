
# Manual Calculator-yodsapat

สร้างไว้ใช้บวกเลขชิลล์ๆ

สร้างตอนเรียน Node.js กับพล

    - สามารถบวกเลขได้
    - ลบได้
    - คูณได้
    - หารยังไม่ได้ทำ

## วิธีการใช้งาน

วิธี require module 

```js
const calculator = require('calculator-yodsapat')

```

### วิธีการบวก

สำหรับการบวก ให้ใช้ function `plus()`

```js
calculator.plus(1,2) //3

```
